# Météo 天气
#### 介绍
一个基于 Vue.js 框架开发的天气应用，气象数据来源于和风天气 api

> Météo 名称来源于法语单词的 `météorologiques` ，也称`météo`，是天气的意思
>

#### 安装教程（Build Setup）

```shell
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build
```
